== Description ==
This is a plugin that provides custom product tab functionality to WooCommerce. You can assign one extra tab to each product and it can be anything you want.

For more detailed info, please visit : http://www.bemoore.com/products/wordpress-plugins-pro/super-product-tabs-pro-for-woocommerce/

This plugin is installed in the standard way.
